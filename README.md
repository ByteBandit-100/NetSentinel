# NetSentinel

Hybrid Network Monitoring + Port Scanning + IDS Tool

## Features (Planned)

- Multithreaded TCP/UDP Scanner
- Banner Grabbing
- Service Detection
- Traffic Analyzer
- Intrusion Detection System
- CLI + GUI Interface
- Logging & Reporting

## Project Structure

Engine → Core Logic  
Interfaces → CLI & GUI  
Logs → Runtime logs  
Reports → Output reports
